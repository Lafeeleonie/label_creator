# Editeur d'etiquettes electronique / Electronic label editor

## Francais

Interface Streamlit pour composer des etiquettes d'atelier electronique et exporter un PDF imprimable aux dimensions exactes.

Les valeurs par defaut sont dans `config.toml` : page A4, marges minimales de `10 mm`, etiquettes de `12,5 x 10 mm` et aucun espacement.

Le nombre de lignes et de colonnes est calcule automatiquement pour ne garder que des etiquettes completes. Si de la place reste sur la page, elle est ajoutee aux marges finales pour centrer la grille.

L'interface utilise un theme sombre par defaut.

La quantite par defaut d'une ligne d'etiquette est `1`.

Les lignes et les reglages courants sont sauvegardes automatiquement dans `.tmp/autosave.json`, un fichier local ignore par git.

Les pages peuvent aussi etre sauvegardees avec un titre, puis rechargees depuis l'interface. Ces sauvegardes locales sont stockees dans `.tmp/saved_pages.json`.

### Lancement

Double-cliquer sur `lancer_streamlit.bat`.

Le script cree ou reutilise `.venv`, installe les dependances et lance Streamlit sur :

```text
http://127.0.0.1:8585
```

### Impression

Le PDF est genere avec ReportLab en millimetres. Dans la fenetre d'impression, choisir `Taille reelle`, `100 %` ou `Aucune mise a l'echelle` pour conserver les dimensions des etiquettes.

L'aperçu integre le PDF reel genere par l'application.

### Symboles

L'interface contient une banque de symboles organisee par familles : passifs, opto/capteurs, diodes, transistors, commutation, alimentation/protection, CI/logique, connecteurs, vis et grandes etiquettes. Elle inclut aussi des composants courants d'atelier comme `Photoresistance`, `Relais reed`, `Zener`, `Arduino`, `ESP32`, `USBasp` ou `ST-Link`. Les symboles de vis sont rendus simplement sous la forme `M2`, `M3`, `M4`, etc., avec aussi les variantes coniques comme `M3 con`.

Sur les etiquettes, le symbole est place au-dessus du texte.

Le formulaire `Custom label` permet d'ajouter une etiquette avec un texte libre dans la zone symbole et un texte libre dans la zone label.

## English

Streamlit interface for composing electronics workshop labels and exporting a printable PDF with exact physical dimensions.

Default values are stored in `config.toml`: A4 page, minimum `10 mm` margins, `12.5 x 10 mm` labels, and no spacing.

The number of rows and columns is computed automatically so only complete labels are kept. If there is unused space on the page, it is added to the final margins to center the grid.

The interface uses a dark theme by default.

The default quantity for a label row is `1`.

Current rows and layout settings are automatically saved to `.tmp/autosave.json`, a local file ignored by git.

Pages can also be saved with a title and loaded again from the interface. These local saved pages are stored in `.tmp/saved_pages.json`.

### Launch

Double-click `lancer_streamlit.bat`.

The script creates or reuses `.venv`, installs dependencies, and starts Streamlit on:

```text
http://127.0.0.1:8585
```

### Printing

The PDF is generated with ReportLab in millimeters. In the print dialog, choose `Actual size`, `100 %`, or `No scaling` to preserve the label dimensions.

The embedded preview shows the real PDF generated by the application.

### Symbols

The interface includes a symbol bank organized by families: passives, opto/sensors, diodes, transistors, switching, power/protection, IC/logic, connectors, screws, and larger labels. It also includes common hobby-workshop components such as `Photoresistance`, `Relais reed`, `Zener`, `Arduino`, `ESP32`, `USBasp`, and `ST-Link`. Screw symbols are rendered simply as `M2`, `M3`, `M4`, etc., with countersunk variants such as `M3 con`.

On labels, the symbol is placed above the text.

The `Custom label` form adds a label with free text in the symbol area and free text in the label area.
